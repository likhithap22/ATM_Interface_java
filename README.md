#  ATM Banking System

The **ATM Banking System** is a simple console-based Java application that simulates basic ATM operations using **Object-Oriented Programming (OOP)** concepts. It provides a user-friendly menu that allows customers to perform essential banking transactions securely.

##  Features

*  Check Account Balance
*  Deposit Money
*  Withdraw Money
*  Validates User Input
*  Prevents Invalid Transactions
* Menu-Driven Console Interface

##  Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Scanner Class
* Methods & Classes

## 📂 Project Structure

* `BankAccount` Class
* `ATM` Class
* `ATMInterface` (Main Class)

## ▶️ How to Run

```bash
javac ATMInterface.java
java ATMInterface
```

## 📋 Sample Operations

* Check Balance
* Deposit Amount
* Withdraw Amount
* Exit the Application
